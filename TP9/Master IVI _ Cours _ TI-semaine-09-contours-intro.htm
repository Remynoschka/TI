TP TI, semaine 09: Détection de contours par approches du premier ordre

Les exercices 1 et 2 utilisent des fonctions pour l'implémentation de macros. Une liste de ces fonctions est disponible à http://rsbweb.nih.gov/ij/developer/macro/functions.html.

L'exercice 3 nécessite l'écriture d'un plugin. Pour cela, on peut se référer à un document de présentation d'ImageJ ou, pour des explications plus approfondies (mais en anglais), au tutoriel de W. Bailer.

Afin de disposer d'un environnement de développement pré-configuré pour Eclipse et d'un ensemble de plugins particulièrement utiles, téléchargez puis décompressez l'archive dans un répertoire que vous créerez sur votre compte (ex. Z:\ImageJ\) et qui sera la racine de votre espace de travail (workspace).

Remarque : vous aurez sans doute besoin de reconfigurer le projet IJ pour le compiler, en réassignant le chemin de tools.jar. Pour cela :

  • Ouvrir les propriétés du projet IJ
  • Dans Java Build Path, onglet Libraries, éditer le chemin de tools.jar, qui devrait se trouver
    • sous Windows, dans C:\Program files\Java\Jdk#.#.#_##\lib
    • sous Mac OS, le fichier même est différent. Il s'agit de /Library/java/home/bundle/Classes/classes.jar
